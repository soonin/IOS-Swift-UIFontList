# IOS-Swift-UIFontList
``` Level : Intermediate ```

This tutorial shows how to make an UITableView of all available iOS system's fonts and diplay them .Yes, There are some predefined system fonts (UIFont.familyNames). This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app To Make TableView of all iOS system Fonts ( UIFont ) 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
